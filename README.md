<!-- GETTING STARTED -->
## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/cr4yfish/taiwan-invasion-checker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run start
   ```

#### Development
To run the project in development mode, run the following command:
```sh
npm run dev
```
